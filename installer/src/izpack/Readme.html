Welcome to ControlTier

This will install and set up ControlTier 3.2 on your system.

ControlTier is a general purpose IT Automation platform, especially useful for application deployment management. The platform is comprised of several components including: CTL, the control dispatcher, JobCenter, web-based tool for executing ad-hoc and defined commands, ReportCenter, centralized activity tracking, Workbench, CMDB and package repository, and Apache JackRabbit, the WebDAV service.

You can find out more about ControlTier on the project website at http://open.controltier.org/.
